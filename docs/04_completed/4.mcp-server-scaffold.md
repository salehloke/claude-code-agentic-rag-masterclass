# Module 1: MCP Server Scaffold + Supabase Verification

✅ **Simple** — Small server skeleton, one tool, registration. Single-pass executable.

## Objective

Build a minimal Python MCP server with stdio transport, register it in Claude Code, and verify end-to-end communication. Confirm Supabase local stack is healthy.

---

## Prerequisites

- [x] Supabase local stack running (`supabase status` shows healthy)
- [ ] Python 3.11+ installed
- [ ] Claude Code CLI available

---

## Tasks

### Task 1: Python Project Setup

- [ ] Create `server/` directory in project root
- [ ] Create venv: `python3 -m venv server/venv`
- [ ] Activate venv and install dependencies:
  ```bash
  cd server
  source venv/bin/activate
  pip install "mcp[cli]"
  ```
- [ ] Create `server/requirements.txt` with pinned versions
- [ ] Add `server/venv/` to `.gitignore`

**Validation**: `pip list | grep mcp` shows the mcp package installed.

---

### Task 2: MCP Server Skeleton

- [ ] Create `server/main.py` with minimal FastMCP server:
  ```python
  from mcp.server.fastmcp import FastMCP

  mcp = FastMCP("rag-masterclass")

  @mcp.tool()
  def hello(name: str = "world") -> str:
      """Say hello — verifies the MCP server is working."""
      return f"Hello, {name}! RAG Masterclass MCP server is running."

  if __name__ == "__main__":
      mcp.run(transport="stdio")
  ```
- [ ] Test locally: `python server/main.py` starts without errors (will hang waiting for stdio input — that's expected)

**Validation**: Server starts without import errors or crashes.

---

### Task 3: Register in Claude Code

- [ ] Create `.mcp.json` in project root (project-scoped, shareable):
  ```json
  {
    "mcpServers": {
      "rag-masterclass": {
        "type": "stdio",
        "command": "server/venv/bin/python",
        "args": ["server/main.py"],
        "cwd": "<project-root-absolute-path>"
      }
    }
  }
  ```
- [ ] Alternatively use CLI: `claude mcp add --transport stdio --scope project rag-masterclass -- server/venv/bin/python server/main.py`
- [ ] Restart Claude Code or run `/mcp` to verify the server is detected

**Validation**: `/mcp` in Claude Code shows `rag-masterclass` server as connected.

---

### Task 4: Verify Supabase Connection Tool

- [ ] Install `supabase` Python client: `pip install supabase`
- [ ] Update `server/requirements.txt`
- [ ] Add a `supabase_health` tool to `server/main.py`:
  ```python
  @mcp.tool()
  def supabase_health() -> dict:
      """Check Supabase local stack connectivity."""
      # Connect to Supabase and run a simple query
      # Returns connection status and available tables
  ```
- [ ] Load Supabase credentials from `.env` using `python-dotenv`
- [ ] Install `python-dotenv`: `pip install python-dotenv`

**Validation**: Call `supabase_health` from Claude Code — returns successful connection status.

---

### Task 5: End-to-End Smoke Test

- [x] From Claude Code, call the `hello` tool — get a greeting response
- [x] From Claude Code, call `supabase_health` — confirms DB is reachable
- [x] Verify both tools show up in Claude Code's tool list

**Validation**: Both tools callable from Claude Code, return expected results.
